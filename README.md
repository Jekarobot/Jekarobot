### 🏦 BHJ-Diploma — Система управления банковскими счетами и транзакциями

#### 📋 Назначение проекта

BHJ-Diploma представляет собой полнофункциональное веб-приложение для управления банковскими счетами и финансовыми операциями. Проект демонстрирует навыки создания клиент-серверного приложения с аутентификацией пользователей, динамическим интерфейсом и REST API. Решение разработано в рамках дипломного проекта и включает как серверную, так и клиентскую части.

#### 🛠 Ключевые технологии и стек

- **Frontend**: Vanilla JavaScript (ES6+), CSS3, HTML5
- **Backend**: Node.js, Express.js
- **База данных**: JSON-файл (db.json) для хранения данных
- **UI Framework**: AdminLTE (Bootstrap-based административная панель)
- **Архитектура**: REST API, модульная структура с разделением на маршруты и контроллеры

#### 🏗 Архитектура и особенности реализации

Приложение построено по принципу SPA (Single Page Application) с использованием компонентного подхода. Клиентская часть включает модули для работы с API, UI-компонентами и маршрутизацией. Серверная часть реализована с использованием Express.js и включает маршруты для управления пользователями, счетами и транзакциями. Особое внимание уделено асинхронной обработке форм, модальным окнам и динамическому обновлению интерфейса без перезагрузки страницы.

#### ✨ Основные возможности

- 🔐 **Аутентификация и регистрация** — безопасный вход и создание новых пользователей
- 💳 **Управление счетами** — создание, просмотр и управление банковскими счетами
- 💸 **Транзакции** — создание и отображение финансовых операций с детальной информацией
- 📊 **Динамический интерфейс** — адаптивная боковая панель с навигацией и виджетом пользователя
- 🖥 **Модальные окна** — интерактивные диалоги для ввода данных и подтверждения действий
- 📱 **Адаптивный дизайн** — поддержка различных устройств через AdminLTE

[🔗 Перейти к проекту](https://github.com/Jekarobot/bhj-diploma)

### 🔥 MyHeat Onboard

**MyHeat Onboard** — это легковесное веб-приложение для управления и мониторинга систем отопления. Проект предоставляет интуитивно понятный интерфейс для быстрой настройки и контроля температуры в помещении. Решение ориентировано на простоту развертывания и минимальное потребление ресурсов.

### 🎯 DOM — Интерактивное веб-приложение для тренировки навыков работы с DOM

#### 📌 Назначение проекта
Проект **"dom"** представляет собой учебное веб-приложение, разработанное для отработки навыков манипуляции DOM-элементами и обработки событий в браузере. Основная цель — создание интерактивного игрового интерфейса с динамическим управлением объектами на странице. Приложение демонстрирует принципы работы с современным JavaScript, модульной архитектурой и сборкой проекта.

#### 🛠 Ключевые технологии и стек
- **JavaScript (ES6+)** — основная логика приложения с модульной структурой
- **Webpack** — сборка проекта (три конфигурации: общая, dev, prod)
- **ESLint** — статический анализ кода с современной конфигурацией
- **HTML5 + CSS3** — семантическая разметка и стилизация интерфейса
- **Yarn** — менеджер зависимостей

#### 🏗 Архитектура и особенности реализации
Проект построен по модульной архитектуре: исходный код находится в папке `src`, где `index.js` служит точкой входа, а `js/app.js` содержит основную логику приложения. Модули вынесены в отдельную директорию `js/modules/`, что обеспечивает чистоту кода и возможность тестирования (папка `js/tests/`). Сборка осуществляется через Webpack с разделением на dev- и prod-конфигурации, что оптимизирует производительность в разных средах.

#### ✨ Основные возможности
- 🎮 Интерактивное взаимодействие с DOM-элементами (изображения гоблина, молота и логотипа Netology)
- ⚡ Динамическое обновление интерфейса без перезагрузки страницы
- 📦 Модульная структура кода для легкого расширения функциональности
- 🧪 Подготовленная инфраструктура для unit-тестирования
- 🚀 Оптимизированная сборка для продакшена с минификацией и бандлингом

### 🚆 Railway Booking

**Railway Booking** — это современное веб-приложение для поиска и бронирования железнодорожных билетов. Проект предоставляет пользователям интуитивно понятный интерфейс для выбора маршрутов, просмотра деталей поездов и оформления заказов.

#### 🛠 Ключевые технологии и стек

- **React 18** с TypeScript для типизированной разработки
- **Vite** для быстрой сборки и горячей перезагрузки
- **ESLint** для поддержания качества кода
- **Feature-Sliced Design (FSD)** — архитектурная методология для масштабируемости

#### 🏗 Архитектура и особенности реализации

Проект построен на **Feature-Sliced Design**, что обеспечивает четкое разделение на слои: `app`, `pages`, `features`, `entities`, `widgets` и `shared`. Используются **React Context** для управления состоянием (провайдеры: `SearchProvider`, `OrderBuildProvider`, `PopupProvider`, `TrainDetailsProvider`), а также кастомные хуки для повторного использования логики. Моковые данные (`mockData`) позволяют тестировать функционал без реального API.

#### ✨ Основные возможности

- 🔍 **Поиск билетов** — выбор города отправления, назначения и даты через компонент `TicketSearch` с кастомным календарем
- 🚃 **Просмотр поездов** — список доступных рейсов с фильтрацией по цене, времени и типу вагона (`FilterAside`, `TicketsList`)
- 📋 **Оформление заказа** — управление пассажирами, выбор мест и оплата через виджеты `Passengers`, `Billing` и `Checkout`
- 🎨 **Адаптивный интерфейс** — слайдеры, попапы и секции (About, HowItWorks, Feedback) для удобства на мобильных устройствах

[🔗 Перейти к проекту](https://github.com/Jekarobot/railway-booking)
